# SSW_Svyrydiuk_TR_KI-308
Курсова робота з предмету "Системне програмне забезпечення"

8 додаткових функцій:

1.Динамічні графіки (Real-time Charts): Візуалізація завантаження CPU та RAM у реальному часі.

2.Менеджер процесів: Список усіх запущених програм із можливістю завершити процес (Kill task) прямо з вашого додатка.

3.Розумні сповіщення: Спливаючі вікна , якщо температура процесора перевищує 80°C або вільна пам'ять стає меншою за 10%.

4.Автоматичне «очищення»: Кнопка (або авто-функція) для завершення фонових процесів, які споживають занадто багато пам'яті.

5.Логування у CSV/JSON: Запис стану системи у файл через певні проміжки часу для подальшого аналізу (історія навантаження).

6.Моніторинг мережі: Відстеження швидкості завантаження та вивантаження даних (Download/Upload speed) для кожного інтерфейсу.

7.Аналізатор дискового простору: Візуалізація того, скільки місця займають різні розділи диску, та виявлення найбільших папок.

8.Контроль автозавантаження: Список програм, що запускаються разом із системою, з можливістю їх відключення.
