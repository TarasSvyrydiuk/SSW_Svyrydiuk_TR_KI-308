# SSW_Svyrydiuk_TR_KI-308
Курсова робота з предмету "Системне програмне забезпечення"

8 додаткових функцій:

Динамічні графіки (Real-time Charts): Візуалізація завантаження CPU та RAM у реальному часі.

Менеджер процесів: Список усіх запущених програм із можливістю завершити процес (Kill task) прямо з вашого додатка.

Розумні сповіщення: Спливаючі вікна , якщо температура процесора перевищує 80°C або вільна пам'ять стає меншою за 10%.

Автоматичне «очищення»: Кнопка (або авто-функція) для завершення фонових процесів, які споживають занадто багато пам'яті.

Логування у CSV/JSON: Запис стану системи у файл через певні проміжки часу для подальшого аналізу (історія навантаження).

Моніторинг мережі: Відстеження швидкості завантаження та вивантаження даних (Download/Upload speed) для кожного інтерфейсу.

Аналізатор дискового простору: Візуалізація того, скільки місця займають різні розділи диску, та виявлення найбільших папок.

Контроль автозавантаження: Список програм, що запускаються разом із системою, з можливістю їх відключення.
